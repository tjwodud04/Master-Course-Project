### HRI_SocialCognition_Avatar  
Code for controlling the virtual avatar in the HRI Social Cognition project  
  
- set up loomie and game mod according to google drive: "HRI Social Cognition fMRI" > "HRI Game Design" > "Setup - Virtual Avatar"  
- requires python 3.7+  
- install packages as set in [requirements.txt](requirements.txt)  
- write your local settings into the [settings.txt](settings.txt)  
- select language (en-US, ko-KR) in [settings.txt](settings.txt) around line 7 ~8  
- go to "HRI_SocialCognition_Avatar" folder and execute "python ." (execute on cmd/terminal etc.)
